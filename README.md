# JavaScript Fundamentals

My 15-day JavaScript fundamentals learning journey with daily beginner-friendly practice files.

## About This Repository

This repository contains 15 days of JavaScript practice projects and examples. Each day focuses on an important beginner JavaScript topic, starting from the basics and gradually moving toward DOM manipulation, events, forms, local storage, and a final mini project.

The goal of this repository is to build a strong foundation in JavaScript through simple, practical, and easy-to-understand examples.

## What I Learned

- What JavaScript is
- How to connect JavaScript with HTML
- Variables and data types
- Operators and expressions
- Strings and template literals
- Numbers and Math methods
- Conditional statements
- Loops
- Functions
- Arrays
- Objects
- DOM selectors
- DOM events
- Form validation
- Local storage
- Building a simple To-Do App
